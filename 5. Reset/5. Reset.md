# 5 Git - Reset

Команда `git reset` передвигает _HEAD_ на указанный коммит,  
при этом могут появиться **недоступные коммиты**.

### Жетский reset _(для сброса)_
```powershell
~~~> git reset
                --hard
                [коммит]
```
+ сбрасывает _индекс_ ;
+ сбрасывает _рабочее дерево_ ;
