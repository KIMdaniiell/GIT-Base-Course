# Git - Основы - Синтаксический сахар

## Синтаксис с тильдами
```powershell
  git show HEAD~~~
```
```powershell
  git show HEAD~3
```

## Синтаксис с собачкой
```powershell
  git show HEAD~3
```
```powershell
  git show `@~3`
```
+ в `powershell` `@` - это включевое слово, поэтому его стоит оборачивать в кавычки ;
